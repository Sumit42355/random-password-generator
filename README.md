# random-password-generator
A simple, user-friendly web-based Random Password Generator that allows users to create secure passwords with customizable options.
*Features

Generate strong random passwords with customizable length (4-32 characters)

Include or exclude uppercase letters, lowercase letters, numbers, and symbols

Copy generated password to clipboard with one click

Reset fields to default settings

Responsive and easy-to-use UI

*Technologies Used

1.HTML - Structure of the webpage

2.CSS - Styling and layout design

3.JavaScript - Logic for generating passwords, handling user interactions

 *How to use

1.Set the desired password length using the input box.

2.Select/deselect checkboxes to include/exclude Uppercase, Lowercase, Numbers, and Symbols.

3.Click the Generate Password button to create a secure password.

4.Click the copy icon to copy the generated password to the clipboard.

5.Click the reset icon to reset the settings to default.
